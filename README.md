## FlipAccounts
A marketplace that provides escrow services to buyers & sellers of socialmedia, freelance & gaming accounts.

### Frontend
- Bootstrap 5
- NextJS
- next-auth for authentication

### Backend
- Python3
- Django
- DjangoRestFramework
- PostgreSQL

### Hosting
- Gunicorn
- Nginx
- Ubuntu 20.04
- Linode shared vps: 1gb ram, 25gb ssd, 1 core
